# ИНСТРУКЦИЯ ПО ЗАПУСКУ СЕРВЕРА

## Запуск:
node server.js

## Доступ:

### С КОМПЬЮТЕРА:
http://localhost:3000

### С МОБИЛЬНОГО (в той же WiFi сети):
http://192.168.0.146:3000

## Остановка:
Ctrl + C в окне сервера

## QR-код для быстрого доступа с мобильного:
Используй любой QR-генератор для: http://192.168.0.146:3000
